# coinmarcap.py
    - A program that is interactive within the terminal that allows the user to view live cryptocurrency data powered by the CoinMarketCap api.

## Getting Started:
    - Copy the repository
    - Rename `.env.example` file to `.env` and add your CoinMarketCap API Key
        - example: `X-CMC_PRO_API_KEY = your-api-key-here`
    - Download dependencies (See Prerequisites)
    - Run `python coinmarcap.py`

## Prerequisites:
    - Python 3.x
    - A CoinMarketCap API Key (free version works)

### Installation:
- `pip install requests`
- `pip install python-dotenv`

## Built with:
- Python
- Requests
- python-dotenv
